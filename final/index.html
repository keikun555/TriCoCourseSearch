How to Import .ics Files to Google Calendar

We are considering an automatic import to google calendar, but for now, please use this guide!

  1. Download .ics file by clicking on
  2. Open Google Calendar on a computer. Note: You can only import from a computer, not a phone or tablet.
  3. In the top right, click Settings > Settings.
  4. Open the Calendars tab.
  5. Click Import calendar between the "My calendars" and "Other Calendars" sections.
  6. Click Choose File and select the file you exported. The file should end in ".ics"
  7. Choose which calendar to add the imported events to. By default, events will be imported into your primary calendar.
  8. Click Import.
